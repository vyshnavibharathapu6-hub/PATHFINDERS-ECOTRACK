# 🌱 EcoTrack – Sustainability Management System

EcoTrack is a web-based application designed to promote sustainability by helping users manage waste, locate recycling centers, and raise environmental complaints efficiently.

---

## 🚀 Features

- ♻️ View nearby recycling centers  
- 📝 Raise environmental complaints  
- 🔍 Track complaint status  
- 📊 Dashboard with activity overview  
- 🔐 (Optional) User authentication  

---

## 🛠️ Tech Stack

- Frontend: React.js  
- Routing: React Router  
- Styling: CSS / Tailwind  
- Backend (optional): Node.js / Express  
- Database (optional): MongoDB / Local Storage

## ⚙️ Installation & Setup

1. Extract the ZIP  
2. Open terminal in project folder  

Install dependencies:
npm install  

Start development server:
npm run dev  

Open browser:
http://localhost:5173  

---

## 🔄 Application Flow

1. User opens the application  
2. Lands on homepage  
3. Chooses an action:
   - View recycling centers  
   - Raise a complaint  
   - Track complaint  
4. Data is processed  
5. Results are displayed  

---

## 📌 Usage

- Use Recycling Centers to find eco-friendly disposal locations  
- Submit issues via Complaint Form  
- Track progress using Complaint ID
